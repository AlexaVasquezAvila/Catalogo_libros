# Catálogo de Libros

## Descripción

Este proyecto académico es una aplicación web desarrollada con Angular que permite administrar un catálogo de libros. La aplicación se encarga de consultar libros, traer su información y marcarlos como favoritos. Además, realiza solicitudes a una API REST externa para obtener información complementaria sobre los libros.

## Contenido del Proyecto

- **Componentes de Angular:**
  - Lista de libros
  - Formulario para añadir libros
  - Edición de libros

- **Servicios:**
  - Conexión con API REST externa

- **Backend:**
  - Implementación sencilla con PHP y MySQL para almacenamiento y consulta de datos.

## Estructura del Proyecto

- **`src/app`**: Directorio principal del proyecto Angular.
  - **`components`**: Componentes visuales (listado y formularios).
  - **`services`**: Lógica para realizar solicitudes HTTP a la API REST.

- **`compendio_api_php`**: Carpeta que contiene el backend en PHP.

## Requisitos para ejecutar el proyecto

- Node.js
- Angular CLI
- Servidor XAMPP (Apache y MySQL)
- Visual Studio Code
- Git

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/AlexaVasquezAvila/Catalogo_libros.git
```

2. Instalar dependencias:

```bash
cd Catalogo_libros/catalogo-libros
npm install
```

3. Ejecutar aplicación:

```bash
ng serve
```
